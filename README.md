Identificador Geracional

Aplicação web desenvolvida para identificar a geração de uma pessoa com base no ano de nascimento informado.
O projeto foi criado com foco no aprendizado de JavaScript básico, manipulação do DOM e lógica condicional.

Funcionalidades

 Entrada do ano de nascimento

 Cálculo automático da geração correspondente

 Exibição dinâmica do resultado na tela

 Interface simples com gradiente e tipografia personalizada

 Tecnologias Utilizadas

HTML5 – estrutura da aplicação

CSS3 – estilização e layout

JavaScript (Vanilla JS) – lógica, funções e manipulação do DOM

🧠 Lógica do Projeto

O usuário informa o ano de nascimento
Ao clicar em Verificar, a função JavaScript é acionada
O sistema analisa o ano informado
A geração correspondente é exibida dinamicamente na página

Exemplo de gerações utilizadas:

Baby Boomers
Geração X
Millennials (Geração Y)
Geração Z
Geração Alpha
